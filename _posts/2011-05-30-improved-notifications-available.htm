Finally it is possible to give your VanillaDesk notifications nicer design.

May release introduces "wiki-like" markdown syntax + customized CSS styling so you can create notifications, which are much easier to read and understand by the recipients.

Supported formatting includes:

  • custom colors
  • bold, italic, underlined font types
  • block quotes
  • headers
  • lists

Well, not completely our invention - we have implemented markdown from Daring Fireball. For detailed information about all possibilities see http://daringfireball.net/projects/markdown/syntax

There are some exceptions which are not supported for e.g. security reasons, e.g. inline HTML. We simply want to keep VanillaDesk usage safe. We hope you'll understand.
