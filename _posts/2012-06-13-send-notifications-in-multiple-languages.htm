--- published: true layout: post title: Send notifications in multiple languages image: notificationLanguages01.PNG summary: You can now send notifications to your customers and users in their own language. ---

If you supply your services to customers in multiple countries, it may not always be appropriate to use English for comunication.
Now, you can define notifications for each language your customers use and VanillaDesk will send the correct version to selected customers.

{% news_img notificationLanguages02.PNG %}

You are able to define a list of languages in the Notification settings that you wish to use.

In a notification template, a tab will be created for each language.
Simply define what the notification should look like in the particular language.

We understand, that configuring templates to your liking will take some time, therefore each language within a template can be activated once you are finished with it.

{% news_img notificationLanguages01.PNG %}
{% news_img notificationLanguages04.PNG %}

The language that is used for a customer is selected in his details. All End users of this customer will receive notifications in this language.

Additionally, each Operator can select which language he wants to receive notifications in.
