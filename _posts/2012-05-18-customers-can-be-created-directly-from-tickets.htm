--- published: true layout: post title: Customers can be created directly from tickets image: newCustomer00.PNG summary: You no longer have to leave a ticket to create a new customer and return to it afterwards.
The process of registering new customers has been made quicker and easier. ---

When creating or editing ticket info, a new customer can be easilly created.

{% news_img newCustomer01.PNG %}

To create a new customer, simply click the (+) button right next to the customer selector field. A modal window will open.

Specify the customers' name and select the services, that will be available to him.
A service-level agreement (SLA) will automatically be created for each of the selected services, saving even more time configuring.
Additionally, you are also able to create multiple locations for the customer (if there are more of them).
